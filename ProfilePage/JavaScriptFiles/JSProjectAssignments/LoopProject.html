Loop, While & Do-While

    Loop

    There are four types of Loop in JS

  1. For

    2. Use to Loop in General array or objects

  2. For...in

    3. Used mostly to loop objects.

  3. For ...of

    4. Used mostly to loop arrays and objects

  4. forEach

    5. Used mostly to loop array with its index number

While & Do-While

While

Is the same as Loop but the difference is that, it keep executing codes till it false and then stops.

Do-While

Also the same as While, but the difference is that, it first executes the code then calculates the condition
